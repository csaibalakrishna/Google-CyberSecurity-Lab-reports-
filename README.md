## About

This repository documents the practical exercises, security investigations, and technical reports completed during the **Google Cybersecurity Professional Certificate**.

Rather than serving as a collection of course materials, this repository has been organized as a cybersecurity portfolio that highlights practical security concepts, analytical thinking, and technical documentation.

The labs cover topics including Identity and Access Management (IAM), phishing investigations, network traffic analysis, denial-of-service attacks, incident response, enterprise security planning, and social engineering.
