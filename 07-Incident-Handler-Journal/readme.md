# Incident Handler's Journal

## Overview

This exercise focuses on documenting a cybersecurity incident using a structured incident journal.

The objective was to record key details of a ransomware incident, summarize the attack, and organize findings using a standardized incident documentation approach.

---

## Objective

- Practice incident documentation.
- Record essential information during a security investigation.
- Summarize the incident using the 5 W's.
- Improve communication and reporting skills during incident response.

---

## Scenario

A healthcare organization experienced a ransomware attack after threat actors gained unauthorized access through a phishing campaign.

Once inside the environment, the attackers encrypted critical systems and demanded payment in exchange for a decryption key.

The exercise involved documenting the incident and identifying important facts that would assist future investigation and response efforts.

## Incident Summary

The documented incident includes:

- **Who:** Organized cybercriminal group
- **What:** Ransomware attack
- **Where:** Healthcare organization
- **When:** Tuesday, 9:00 AM
- **Why:** Initial access gained through a phishing attack, followed by ransomware deployment for financial gain.

---

## Root Cause

The attackers successfully compromised the organization through a phishing attack.

After obtaining access, they deployed ransomware that encrypted critical organizational data and demanded a ransom payment for decryption.

---

## Response Considerations

Following documentation of the incident, several important questions remain:

- How can similar incidents be prevented in the future?
- Should the organization negotiate or pay the ransom?
- What containment and recovery actions should be prioritized?
- How should evidence be preserved for forensic analysis?

---

## Recommended Security Controls

To reduce the likelihood of similar incidents:

- Deploy Multi-Factor Authentication (MFA).
- Conduct regular phishing awareness training.
- Maintain offline and tested backups.
- Apply timely security patches.
- Implement Endpoint Detection and Response (EDR).
- Restrict user privileges using the Principle of Least Privilege.
- Continuously monitor network activity.

## Key Takeaways

Accurate documentation is a critical component of every cybersecurity investigation.

A structured incident journal helps responders record essential information, maintain a timeline of events, support forensic investigations, and improve organizational response to future security incidents.
