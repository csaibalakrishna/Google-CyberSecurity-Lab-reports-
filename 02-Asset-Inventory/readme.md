# Asset Inventory and Classification

## Overview

This lab focuses on identifying organizational assets, documenting their characteristics, and classifying them based on sensitivity and access requirements.

Maintaining an accurate asset inventory is a fundamental cybersecurity practice that enables organizations to understand what assets they own, determine their business value, and apply appropriate security controls.

---

## Objective

The objective of this exercise was to:

- Identify different organizational assets.
- Record ownership and location information.
- Assess the sensitivity of each asset.
- Assign appropriate access classifications based on business requirements.

## Assets Identified

The inventory included multiple types of assets such as:

- Network Router
- Desktop Computer
- Guest Smartphone
- External Hard Drive
- Streaming Media Device
- Portable Gaming Console

Each asset was documented with:

- Network access frequency
- Asset owner
- Physical location
- Security notes
- Sensitivity classification

---

## Asset Classification Levels

The exercise categorized assets according to their sensitivity:

| Classification | Description |
|----------------|-------------|
| Public | Information that can be shared without restrictions. |
| Internal-Only | Information intended for internal organizational use. |
| Confidential | Sensitive information requiring restricted access. |
| Restricted | Highly sensitive information accessible only on a need-to-know basis. |

---

## Key Findings

The inventory demonstrates that different assets require different security controls depending on:

- Business importance
- Information stored
- Ownership
- Exposure to external networks
- Potential security risks

Proper asset classification helps organizations prioritize protection efforts and implement appropriate access controls.

---

## Security Importance

A well-maintained asset inventory enables organizations to:

- Identify critical assets
- Protect sensitive information
- Improve risk assessments
- Support incident response activities
- Simplify security audits
- Meet compliance requirements

Without accurate asset inventories, organizations may overlook vulnerable systems or fail to secure valuable resources.

This exercise reinforced the importance of maintaining an accurate asset inventory as part of an organization's cybersecurity program.
